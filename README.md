# iLuminIA---Artefatos-da-Pesquisa
Este repositório contém os artefatos utilizados no estudo sobre geração automatizada de Planos Educacionais Individualizados (PEIs) com apoio de LLMs e RAG. São disponibilizados os instrumentos de avaliação, documentos utilizados no experimento e demais materiais necessários para apoiar a reprodutibilidade dos resultados apresentados no artigo.
