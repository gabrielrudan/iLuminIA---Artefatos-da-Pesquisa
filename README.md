# Artifact Repository

Este repositório disponibiliza os artefatos utilizados no estudo submetido para avaliação científica, cujo objetivo é investigar o uso de Modelos de Linguagem de Grande Escala (LLMs) apoiados por Retrieval-Augmented Generation (RAG) na geração automatizada de Planos Educacionais Individualizados (PEIs).

## Objetivo

O propósito deste repositório é fornecer acesso aos materiais utilizados durante o processo de avaliação da proposta, permitindo maior transparência e reprodutibilidade dos resultados apresentados no artigo.

## Estrutura do Repositório

### `/evaluation-form`

Contém o formulário utilizado na avaliação pedagógica dos PEIs.

### `/pei-documents`

Contém os quatro PEIs utilizados no experimento de avaliação cega, identificados apenas como PEI A, PEI B, PEI C e PEI D.

### `/results`

Contém tabelas, gráficos e demais artefatos produzidos durante a análise dos resultados.

## Processo de Avaliação

A avaliação pedagógica foi conduzida por especialistas da área educacional por meio de um processo de avaliação humana cega. Os participantes analisaram quatro PEIs anonimizados e responderam questões relacionadas aos seguintes critérios:

* Clareza das informações;
* Adequação pedagógica;
* Adaptação às necessidades do estudante;
* Coerência das estratégias propostas;
* Viabilidade de aplicação no contexto escolar inclusivo.

## Observação

Para preservar o processo de revisão duplamente cega, informações que possam identificar autores, instituições ou repositórios do projeto principal foram removidas deste material. O código-fonte da plataforma não é disponibilizado neste repositório devido a restrições contratuais relacionadas ao desenvolvimento da solução.
